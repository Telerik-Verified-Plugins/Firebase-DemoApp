Firebase Plugin Demo

The Firebase plugin allows you to easily install and use the Firebase JavaScript API.

With Firebase you get an easy to set up cloud JSON database which supports realtime data sync between all your clients.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android/WP - App package.

Initialization

Before interacting with it, you need to open a connection to your cloud database. In a real app you'll do this immediately after 'deviceready' has fired.

Reading data

Firebase immediately returns any current value and watches this object for future changes.
